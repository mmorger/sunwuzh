INTRODUCTION
------------

This module makes destination anchor links that start with a hashtag scroll 
users to the destination instead of the browser default jump behavior.

REQUIREMENTS
------------

This module does not have any dependency on any other module.

INSTALLATION
------------

Install as normal (see http://drupal.org/documentation/install/modules-themes).

1. Unpack the Scroll to destination anchors folder and contents in the
   appropriate modules directory of your Drupal installation.  This is
   usually sites/all/modules/
2. Enable the Scroll to destination anchors module in the administration
   tools.

CONFIGURATION
-------------
Add destination anchor links to the content of your website. The format of 
these is <a href="#id_of_where_you_want_to_scroll">Click here</a>.
